serialization:-
---------------------- 
serialization helps us to save or retrieve state of an object
basiucally serilization is two steps 
1.serialization :- to convert object state to some internal object representation.
2.deserialization :- reverse .convert internal representation to object.

Two important methods
-----------------------------------
ObjectOutputStream.writeObject()//serialize and write to file
ObjectInputStream.readObject()// read from file and deserialize

if any member variable in that object if you don't want serilization then make that memeber variable as a transient.

how do you serialize an object using serializable interface
-----------------------------------------------------------------------

if any  object that  implements serializable interface.then that object is serializable.

	objects in bytes(binay format )lo convert chesi file lo store cheyyadam= serialization by using writeObject().
	
	binay data to object ga convert cheyali ante = deserialization--readObject()


examples:-
---------------

Class Rectangle implemenst Serializable{
public Rectangle(int length,int breadth){
this.length = length;
this.breadth = breadth;
area = length * breadth;
}
int length;
int breadth;
int area;
}
Q) How do you serialization an object in java?
A) FileOutputStream fileOutputstream = new FileOutputstream("Rectangle.ser");
  ObjectOutputStream output = new ObjectOutputStream(fileOutputstream);
		 output.writeObject(new Rectangle(5,6)); 
         output.close();


Q) Deserialize an java object?
A)FileInputStream fileInputstream = new FileOutputstream("Rectangle.ser");
  ObjectInputStream objectInputStream = new ObjectInputStream(fileInputstream);
Rectangle rectangleObj = 		(Rectangle) objectInputStream.readObject(); 
         objectInputStream.close();

Syatem.out.println(rectangleObj.length);//5
Syatem.out.println(rectangleObj.breadth);//6
Syatem.out.println(rectangleObj.area); // 30

remember:- the object that you want to serialize should need to implement Seralizable interface.
