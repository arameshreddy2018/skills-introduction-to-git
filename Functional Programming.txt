Q)what is streams?
A) A stream is a source of objects.in above example we created stream as  alist.

Streams have intermediate operations and Terminal Operations.in the example above,we used filter as intermediate operation and reduce as a terminal operation.
in streams we can use lot of  intermediate operations and one Terminal Operations

3.sorted
